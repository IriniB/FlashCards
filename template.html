Flash cards

Аннотация

Назначение:
Помощь студентам при подготовке к элементам контроля, путем облегчения процесса запоминания информации.
Целевая аудитория:
Студенты, учащиеся школы.

Варианты использования 1. Создание карточки
2. Просмотр всех созданных карточек
3. Вывод ответа на вопрос
4. Сокрытие ответа на вопрос
5. Сокрытие ответов для всех карточек
6. Возможность редактировать вопрос и ответ на него
7. Сохранение всех созданных карточек в файл
8. Загрузка карточек из файла
Сценарии использования

Создание карточки:
Пользователь нажимает на кнопку Create card
Загружается форма создания карточки
Пользователь может просто вернуться к основной странице, нажав кнопку Close
Пользователь записывает вопрос в поле Question
Пользователь записывает ответ на вапрос в поле Answer
Пользователь нажимает на кнопку Save
Происходит переход на основную страницу, в которой уже добавлена созданная карточка

Просмотр всех карточек:
На основной странице находятся все созданные карточки со скрытыми ответами
Пользователь пытается ответить на вопрос
Пользователь нажимает на кнопку Show answer на соответствующей карточке
Отображается поле с ответом, чтобы пользователь смог проверить себя
Пользователь может нажать на кнопку Hide answer на соответствующей карточке,
чтобы снова скрыть ответ на вопрос
Если поьлзователь хочет начать сначала, он наживает на кнопку Hide all answers и ответы
во всех карточках снова скрываются

Редактирование карточки:
На основной страние отображаются все карточки, вопросы и ответы в которых можно редактировать
Для сохранения достаточно скачать файл.
--или--
Пользователь скачивает файл
Редактирует вопросы и ответы в файле
Пользователь загружает файл в приложение

Сохранение карточек:
На основной странице пользователь нажимает на кнопку Download
После чего скачивается csv файл с карточками.
Структура файла: он состоит из 2-х столбцов. В первом расположены вопросы,
во втором - ответы на них. Каждая карточка представляется одной строчкой

Загрузка карточек:
Пользователь нажимает на кнопку Выберите файл
Открывается окно, в котором пользователь выбирает нужный файл
После подтверждения своего выбора пользователь нажимает на кнопку Upload
Все карточки, которые были описаны в файле появляются на основной странице
